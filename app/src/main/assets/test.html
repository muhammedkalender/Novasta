WP Sadece Kayıtlı Kullanıcılar Görebilir

19 Haziran 2018/in Web Yazılım Makale /tarafından novasta

WordPress üzerinden basit bir kod yazımı ile login sayfası değiştirilip, sadece kayıtlı kullanıcıların görebileceği web sayfasının yapılması için;

// Redirect users who arent logged in...
function members_only() {
global $pagenow;
// Check to see if user in not logged in and not on the login page
if( !is_user_logged_in() && $pagenow != 'wp-login.php' )
auth_redirect();
}
add_action( 'wp', 'members_only' );
